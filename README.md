# Learn-OpenGL

Learn OpenGL from [learnopengl](https://learnopengl.com/)

Original Code repository: https://github.com/JoeyDeVries/LearnOpenGL

## Environment

```
vscode + msvc + cmake + vcpkg
```

## Requirement

```
glfw3
glad
glm
assimp
```

## Run

```
cd [Class-folder]
mkdir build
cd build
cmake ..
cmake --build .
```
